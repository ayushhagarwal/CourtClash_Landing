# CourtClash Landing Page

A minimal, premium landing page for the CourtClash iOS app built with Angular and Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm start
```

3. Open browser at `http://localhost:4200`

## Build

```bash
npm run build
```

## Features

- Angular 17 standalone components
- Tailwind CSS for styling
- Responsive design
- Clean, Apple-like minimal aesthetic
- Privacy and Contact pages
